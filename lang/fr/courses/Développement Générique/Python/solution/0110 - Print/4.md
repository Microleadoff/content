
### Énoncé :

Écrivez un programme qui affiche "5 + 10 = 15" en utilisant la fonction `print()`.

### Contrainte :

Utilisez les variables `a = 5` et `b = 10` pour calculer et afficher le résultat.

### Résultat Attendu :

Le programme affiche : "5 + 10 = 15".

### Solution :

```python
a = 5
b = 10
print(f"{a} + {b} = {a + b}")
```
